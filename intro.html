What Is Web Development?

Starting Out In Web Development.

Web development is the work involved in developing a web site for the Internet (World Wide Web) or an intranet (a private network). Web development can range from developing a simple single static page of plain text to complex web-based internet applications (web apps), electronic businesses, and social network services. A more comprehensive list of tasks to which web development commonly refers, may include web engineering, web design, web content development, client liaison, client-side/server-side scripting, web server and network security configuration, and e-commerce development. Among web professionals, "web development" usually refers to the main non-design aspects of building web sites: writing markup and coding. Most recently Web development has come to mean the creation of content management systems or CMS. These CMS can be made from scratch, proprietary or open source. In broad terms the CMS acts as middleware between the database and the user through the browser. A principle benefit of a CMS is that it allows non-technical people to make changes to their web site without having technical knowledge.

Who Can Learn Web Development?

Anyone can learn to be a web developer. All it takes to become a web developer is the desire to learn something new. When I first gave thought to the web development industry, I didn't think I would be able to learn due to coding being to complex. I was wrong! Over time I have come to learn that it is possible for anyone if they are willing to put in the time and effort. I have met web development student from children all the way up to seniors. It just takes time and dedication.

What Are We Learning?

There are many different aspects to web development as I have mentioned above. In my Into To Web Dev course I learned many things. Html and CSS are the basics to start off learning first. Html is like the skeleton of the website. The CSS is like the visual features that make the website look the way it does. Javascript is the functionality of the website. We can also make really easily responsive websites or web apps using a framework called Bootstrap. There are many different programming languages out there such as Java, Python, React, Angular, and Vue. These are more advanced programming languages.

Types Of Web Development

What Is Front End Development?

Front end web development refers to the UX/UI (User Interface). This is the design and creation of the website or web application that the user can see and interact with directly. This can be made possible using Html, CSS, Javascript and other frame works such as ReactJS, Angular, and Vue. Web API's can also be utilized to recall information that is needed in Front End Development.

What Is Back End Development?

Back end web development consists of the server side of web development. It is all the stuff that goes on behind the scenes of a website. It can consist of data storage, security, and API's. Back end coding is code that helps the servers communicate with websites to store, transfer and delete data. Back end developers are also responsible for making sure the back-end performs quickly and responsively to user requests from the front end. Some popular back end languages include, SQL, Python, Java, C# and Node.js

What Is Full Stack Development?

Full stack developers are able to develop both front end and back end software. They combine the front end development in Html, CSS, Javascript and other languages with back end software created with SQL, Java, Python, and many others. Some advantages of being a full stack developers are, you are able to prototype very quickly. Also you can switch between front end and back end depending on what is needed. On the downside being a full stack developer can be very complex.
